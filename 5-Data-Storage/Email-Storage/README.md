# Email Storage

**From PRD Section 3.3.1**

Store email addresses and track scan usage.

---

## Storage Details

- Storage: Google Spreadsheet (via N8N)
- Tracking: Email-based only (no cookies/localStorage)
- Initial scans: 3 per email
