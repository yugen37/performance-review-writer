# OpenAI Integration

**From PRD Section 3.4.1**

Generate enhanced rewrite using trained prompt via N8N OpenAI Node.

---

## Configuration

- **API Key:** Stored in N8N environment variables
- **Model:** GPT-4 or GPT-3.5
- **Temperature:** 0.7
- **Max Tokens:** 300
- **Prompt:** Trained prompt (provided by user)

---

## Files to Store Here

- OpenAI prompt files
- API configuration
- Cost tracking
- Error handling documentation
- Prompt testing results
