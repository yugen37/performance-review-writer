# Edge Cases

**From PRD Section 2.2**

## Edge Cases

- User pastes empty text → Show error
- User pastes text > 200 words → Show error
- User has used 3 scans → Show upgrade prompt
- API fails → Show error message, allow retry
- Invalid email format → Show validation error
- User closes email gate → Can reopen
